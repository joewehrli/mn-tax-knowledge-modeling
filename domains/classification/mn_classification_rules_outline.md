
# Minnesota Property Classification – Rule Outline

## 1. Core Legal Structure (MN Statutes §273.13)
- **Class 1:** Residential Homestead
  - 1a – Homestead
  - 1b – Disabled/Blind Homestead
  - 1c – Ma-and-Pa Resort
- **Class 2:** Rural & Agricultural Land
  - 2a – Agricultural
  - 2b – Rural Vacant
  - 2c – Managed Forest
- **Class 3:** Commercial/Industrial
- **Class 4:** Residential Non-Homestead
  - 4a – Apartment
  - 4b – Non-homestead residential
  - 4bb, 4d(1), 4d(2) – Low-income housing
- **Class 5+:** Miscellaneous (e.g., unmined ore, railroads)

## 2. Manual Rules (ARL Module 3)
- Eligibility thresholds (e.g. 10-acre minimum for 2a)
- Split-classification procedures
- Homestead filing and verification steps
- Classification based on use as of January 2

## 3. Legislative and Bulletin Updates
- New classes (e.g., 4d(1), 4d(2))
- Rule changes published annually in DOR bulletins

## 4. Administrative Rules & Precedents
- Clarification of “contiguous,” “agricultural use,” etc.
- Court opinions interpret ambiguous cases
