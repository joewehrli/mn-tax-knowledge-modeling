
# Data Requirements for MN Property Classification Evaluation

## A. Parcel Data
- Parcel ID, acreage
- Legal description, parcel boundaries
- GIS shapefile for geometry

## B. Land Use Data
- Land use code (current and previous year)
- Type of agricultural activity (crop, pasture, etc.)
- Structure usage type (residential, commercial, rental)

## C. Ownership & Homestead Data
- Owner name and address
- Residency verification (MN primary residence)
- Disability/blindness status (for 1b)

## D. Program Participation
- Enrollment in Green Acres or 2c programs
- Low-income housing certification (4d)
- Relevant declarations filed with county

## E. Valuation & Tax Data
- Estimated market value
- Class rate (by classification)
- Net tax capacity calculation

## F. Declaration/Certification Data
- Homestead application status and date
- 4d housing filing date and approval
- Agricultural use affidavit or similar

## G. Temporal Context
- Use and ownership status as of January 2
- Compliance deadlines (e.g., October 1, May 1)

## H. GIS & Spatial Relationships
- Contiguity analysis (for 2a farmland)
- Distance to other parcels under same ownership
